# Sicherheitsrichtlinie

Unterstützt wird jeweils die **neueste veröffentlichte Version** der
Ductus-Pakete (npm-Scope [`@ductus`](https://www.npmjs.com/org/ductus),
pub.dev-Paket [`ductus`](https://pub.dev/packages/ductus)).

## Eine Sicherheitslücke melden

Bitte melde Sicherheitslücken **nicht** als öffentliches Issue, sondern
vertraulich über GitHubs private Schwachstellenmeldung:

**[Report a vulnerability](https://github.com/PlaxXOnline/ductus/security/advisories/new)**

Beschreibe möglichst konkret, wie sich das Problem reproduzieren lässt und
welche Pakete/Versionen betroffen sind. Du bekommst zeitnah eine Rückmeldung;
bitte veröffentliche Details erst nach einer Behebung (koordinierte Offenlegung).

*English: please report vulnerabilities privately via the advisory form
linked above — not as public issues.*
