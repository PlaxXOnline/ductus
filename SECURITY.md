# Security policy

The **latest published version** of the Ductus packages is supported
(npm scope [`@ductus`](https://www.npmjs.com/org/ductus),
pub.dev package [`ductus`](https://pub.dev/packages/ductus)).

## Reporting a vulnerability

Please do **not** report vulnerabilities as public issues; report them
confidentially via GitHub's private vulnerability reporting:

**[Report a vulnerability](https://github.com/PlaxXOnline/ductus/security/advisories/new)**

Describe as concretely as possible how the problem can be reproduced and
which packages/versions are affected. You will receive a timely response;
please do not publish details until a fix is available (coordinated
disclosure).
