# Ductus docs website (Starlight)

This directory is an Astro/Starlight website generated by Ductus; the content under `src/content/docs/` is generated by Ductus. Install the dependencies once with `npm install`. After that, `npm run dev` starts a local preview and `npm run build` produces the static website under `dist/`.

## Mermaid diagrams

Code blocks with the language `mermaid` — Ductus generates one journey diagram
(“main path”) and one flowchart (“flow diagram”) per flow — are rendered as
diagrams in the browser:

- A remark plugin in `astro.config.mjs` converts the code blocks into
  `<pre class="mermaid">` at build time (applies to Markdown and MDX).
- A small inline script loads Mermaid at runtime via CDN
  (jsdelivr, mermaid@11) and renders all diagrams to match the selected color
  scheme — including live switching between light and dark.
- **Offline fallback:** without an internet connection only the CDN import
  fails; the diagram source then remains visible as a readable code block and
  the page keeps working.
